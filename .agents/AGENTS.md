## Backend Phase Tracking

When working on backend tasks, you MUST keep the backend phase markdown files (e.g. `md/backend/phaseX_*.md`) up to date.
As you or the user complete features, update the corresponding tasks in these markdown files line-by-line by checking them off (changing `- ` to `- [x] `). Maintain synchronization with the team by marking each subtask individually as completed.
